# 旅行规划 AI 工作流提示词

将下面这段提示词粘贴给 Claude，替换 `[目的地]`、`[出发城市]`、`[出发日期]` 即可。

---

## 主提示词（直接复制使用）

```
你是我的专属旅行规划助手，请帮我规划一次从 [出发城市] 到 [目的地] 的旅行，出发日期是 [出发日期]，共 [天数] 天。

请按以下步骤依次使用工具收集信息，再统一整合：

### 第一步：查询交通方案（12306-mcp）
- 查询从 [出发城市] 到 [目的地] 的火车/高铁票
- 找出出发当天最优时间段的车次（上午/下午各一个选项）
- 记录车次号、出发时间、到达时间、历时、票价（二等座/一等座）

### 第二步：收集小红书攻略（xhs-mcp）
- 搜索关键词：「[目的地] 旅游攻略」
- 搜索关键词：「[目的地] 必去景点」
- 搜索关键词：「[目的地] 美食推荐」
- 搜索关键词：「[目的地] 住宿推荐」
- 整理出：前5个景点、前5个餐厅、前3个住宿推荐、常见避坑提示

### 第三步：规划路线（amap-mcp）
- 以住宿地点为中心，规划每天的游览路线
- 查询景点之间的交通方式和时间
- 确保路线合理（地理上就近，避免来回跑）
- 估算每天的交通费用

### 第四步：整合输出
请将以上信息整合，输出以下内容：
1. **交通方案**：推荐车次 + 备选车次
2. **逐日行程**：每天从早到晚的安排（时间 | 地点 | 活动 | 预计费用）
3. **餐饮推荐**：按每天分配推荐餐厅
4. **住宿建议**：推荐区域和具体酒店/民宿
5. **费用估算**：交通 + 景点门票 + 餐饮 + 住宿 的总预算
6. **实用提示**：当地注意事项、避坑指南、必备APP

最后，请将整合结果转换为 HTML 旅行规划表（使用 travel-template.html 中的格式）。
```

## 快捷版提示词（简洁模式）

```
帮我规划 [出发城市]→[目的地] 的 [天数] 天旅行，[出发日期] 出发。
请依次调用：
1. 12306-mcp 查火车票
2. xhs-mcp 搜「[目的地]旅游攻略」「[目的地]美食」
3. amap-mcp 规划每日路线

整合后输出完整行程表和费用预算。
```
